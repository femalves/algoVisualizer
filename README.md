## :runner: How to run

### :point_up: Frontend

```bash
# Clone this repository
$ git clone https://github.com/femalves/algoVisualizer

# From the terminal, go to folder
$ cd visualizer

# Install dependencies
$ npm install or yarn add

# Run Application
$ npm start ou yarn start

# Go to http://localhost:3000
```
